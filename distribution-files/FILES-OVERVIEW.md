## 📋 **インストール推奨順序**

### **1️⃣ 事前準備**
```
□ Windows 10/11 PC を準備
□ 管理者権限があることを確認
□ USB容量 3GB以上 を確認
```

### **2️⃣ ベースソフトウェア**
```
□ OllamaSetup.exe を実行
□ PC再起動
```

### **3️⃣ モデル配置**
```
□ ollama-models/ の中身をコピー
□ %USERPROFILE%\.ollama\models\ に配置
```

### **4️⃣ アプリインストール**
```
□ Medical Record LLM-1.0.0-Setup.exe を実行
□ デスクトップショートカット作成
```

### **5️⃣ 動作確認**
```
□ アプリ起動
□ ユーザー登録
□ 音声入力テスト
□ 医療AI機能テスト
```

## 🏥 **病院配布時の注意点**

### **✅ 推奨配布方法**
- **USB配布**: 32GB以上のUSBメモリ
- **ネットワーク共有**: 院内共有フォルダ
- **CD/DVD**: 容量不足のため非推奨

### **⚠️ セキュリティ注意**
- すべてオフライン動作（インターネット不要）
- 患者データは外部送信されません
- 院内ネットワークのみで動作
